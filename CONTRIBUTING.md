# Contributing to the Interactive ML Study Guide

Thank you for helping improve this resource for students worldwide.

## Before You Submit

- Ensure the resource is **free** (no paywalls, no mandatory sign-up)
- Ensure the resource is **primarily visual or interactive** — static
  text explanations belong elsewhere
- Search [open issues](../../issues) and [existing entries](README.md)
  to avoid duplicates
- The resource should help a student understand *how* an algorithm
  works, not just reference its definition

## How to Add a Resource

1. Fork this repository
2. Add your entry to the appropriate section in `README.md`
3. Follow the existing format:
   `- **[Name](URL)** — One or two sentence description.`
4. Open a Pull Request with a title like:
   `Add [Resource Name] to [Section Name]`
5. In the PR description, briefly explain why this resource meets
   the quality bar

## Format Guidelines

- Use bold for the resource name
- Keep descriptions to 1–2 sentences
- Focus the description on *what a student can do* with the resource,
  not just what it is
- Alphabetical order is not required — place entries where they
  fit most logically

## What Will Be Declined

- Paid tools or tools with feature-gated free tiers
- Static blog posts or articles (however good)
- Resources not specifically relevant to university-level ML study
- Self-promotional submissions without genuine educational value
